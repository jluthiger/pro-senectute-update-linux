Warum sind wir heute hier?

  • Windows 10 Support endet im Oktober 2025
  • Viele Computer können Windows 11 nicht installieren
  • Ein neuer Computer kostet schnell 800-1500 CHF
  • Es gibt eine Alternative!

Was ist Linux?

  • Ein Betriebssystem wie Windows oder macOS
  • Entwickelt seit 1991
  • Kostenlos und frei verfügbar
  • Wird von Millionen Menschen weltweit genutzt

Wo wird Linux bereits eingesetzt?

  • Android-Smartphones basieren auf Linux
  • Die meisten Webserver im Internet
  • Smart-TVs und Router
  • Supercomputer und Raumfahrt

Linux ist also kein Experiment - es ist bewährt!

Linux Mint - Unser Favorit

  • Speziell für Einsteiger entwickelt
  • Sieht Windows sehr ähnlich
  • Einfache Bedienung
  • Grosse deutschsprachige Community

Was bedeutet ein Umstieg?

  • Ihr Computer bleibt derselbe
  • Windows wird durch Linux ersetzt
  • Ihre Dateien können übernommen werden
  • Einige Programme ändern sich

Was bleibt gleich?

  • Internet surfen (Firefox, Chrome)
  • E-Mails schreiben (Thunderbird)
  • Fotos anschauen und verwalten
  • Musik und Videos abspielen
  • Dokumente bearbeiten (LibreOffice)

Was ändert sich?

  • Aussehen ist etwas anders (aber ähnlich)
  • Microsoft Office wird zu LibreOffice
  • Einige Windows-Programme laufen nicht
  • Neue Wege, Programme zu installieren

Vorteile von Linux

1. Kostenlos

  • Kein Kaufpreis für das Betriebssystem
  • Keine Lizenzgebühren
  • Kostenlose Office-Programme enthalten
  • Tausende weitere Programme gratis

2. Sicher

  • Weniger Viren und Schadsoftware
  • Regelmässige Sicherheitsupdates
  • Kein teures Antivirenprogramm nötig
  • Ihre Daten gehören Ihnen

3. Schnell & Ressourcenschonend

  • Läuft auch auf älteren Computern flüssig
  • Startet schneller als Windows
  • Wird nicht langsamer mit der Zeit
  • Verlängert die Lebensdauer Ihres Computers

4. Unabhängigkeit

  • Keine Abhängigkeit von Microsoft
  • Keine erzwungenen Updates
  • Keine Werbung im Betriebssystem
  • Volle Kontrolle über Ihren Computer

5. Datenschutz

  • Keine Datensammlung durch Microsoft
  • Sie bestimmen, was gesendet wird
  • Transparenz durch offenen Quellcode
  • Respekt für Ihre Privatsphäre

Nachteile von Linux

1. Umgewöhnung nötig

  • Neue Oberfläche (aber ähnlich zu Windows)
  • Einige Dinge funktionieren anders
  • Kurze Einarbeitungszeit notwendig

2. Einige Programme fehlen

  • Microsoft Office nicht verfügbar (aber LibreOffice)
  • Adobe Photoshop nicht verfügbar (aber GIMP)
  • Einige Bankprogramme funktionieren nicht
  • Manche Spiele laufen nicht

3. Hardware-Kompatibilität

  • Manche Drucker/Scanner brauchen Anpassungen
  • Spezielle Hardware kann problematisch sein
  • Vorher testen ist empfehlenswert

4. Weniger bekannt

  • Familie/Bekannte kennen sich evtl. nicht aus
  • Support meist über Internet/Community
  • Weniger "offizieller" Telefon-Support

Was kostet der Umstieg?

Option 1: Selbst installieren

Kosten: 0 CHF

  • Linux ist kostenlos
  • Anleitungen im Internet verfügbar
  • USB-Stick (ca. 10 CHF) benötigt

Option 2: Mit Hilfe

Kosten: ca. 150 CHF

  • Datensicherung inklusive

Zeitlicher Aufwand

Installation

  • Datensicherung: 1-2 Stunden
  • Linux-Installation: ca. 30-60 Minuten
  • Programme einrichten: 1-2 Stunden
  • Insgesamt: ca. einen halben Tag

Einarbeitung

  • Grundbedienung: wenige Stunden
  • Routiniert werden: 1-2 Wochen
  • Die meisten Dinge funktionieren sofort vertraut

Tipp: Einfach ausprobieren und keine Angst haben!

Wie kann ich Linux ausprobieren?

Option 1: Live-USB

  • Linux vom USB-Stick starten
  • Ohne Installation testen
  • Windows bleibt erhalten
  • Risikofrei ausprobieren

Option 2: Virtuell testen

  • Linux in einem Fenster unter Windows
  • Mit VirtualBox (kostenlos)
  • Beide Systeme gleichzeitig

Option 3: Direkt installieren

  • Erst Datensicherung machen!
  • Linux ersetzt Windows komplett
  • Oder: Dual-Boot (beide Systeme)
